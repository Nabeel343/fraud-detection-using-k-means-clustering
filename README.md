# Fraud Detection using K-Means Clustering

## Project Overview
This project detects fraudulent transactions using unsupervised machine learning (K-Means Clustering) and provides a Streamlit dashboard for real-time prediction.

## Features
- Data generation and preprocessing
- K-Means clustering for anomaly detection
- Risk scoring system
- Streamlit web application
- Data visualization (PCA, cluster plots)

## Tech Stack
Python, Pandas, Scikit-learn, Streamlit, Matplotlib
